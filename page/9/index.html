前言

  1. Done is better then perfect. 不管怎么样，先上一个能用的博客再说，让自己把博客先更新起来。Hexo 的开发者很多，功能够用，也集成了很多第三方插件。
  2. Keep it Simple and Stupid. 很喜欢 Hexo 里的一个主题，简洁大方，相当符合这个原则，而之前在 jekyll 里没看到。
  3. Hexo 基于 node 运行，更有助于熟悉通用的 Web 开发。
  4. 静态网站不用折腾搞 SEO 了。
