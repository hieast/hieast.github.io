2019年王道单科计组-内容讨论

发表于 2018-04-22 |

前言

准备把机器学习基础学一遍，顺便考个研。有点耽误赚钱，不过我还年轻。

这篇文章记录做 2019 年王道单科计组那本书的时候的一些疑问，有书里的错误，也有我的肤浅，希望和各位读者探讨考究一番。

我之前看过一次 CSAPP，做过练习题。这次大概花了 7 周的业余时间，跟着中国大学 MOOC 上刘宏伟老师和袁春风老师的视频和袁老师的教材过完了王道单科一轮，还剩 5~7 章的部分大题没做。

这门课给我最大的感受就是术语不统一，不同的教材（老师）对于术语的使用和少数概念的划分都有所不同。因此在学习过程中需要博采众长，真正理解计算机不同部分的功能和组成之间的关系，把术语进行归类，牢记同义术语和英文原文，做到以不变应万变。

建议第一次学的同学到中国大学 MOOC 去看刘宏伟老师的计组视频，课后习题质量很高。

内容上的疑问

  1. p31.一.16 CRC 码和海明码都能发现两位错误并纠正一位错的编码。
  2. p71.一.29 答案解析中 “右移一位变成 11101” 应为 “右移一位变成 11001” 。
  3. p73.二.3 答案解析中的指数用二进制表示，缺少标记。
  4. p74.二.10 第一问答案解析是基于 n=0 的情况，但是题意也可以理解为 n 可以取任意值。我倾向于后者，考察的知识点更多。
  5. p85 表2-9 无符号定点小数，最前面不应该有小数点，更不应该为 0，因为没有符号位。后面一列最大值也算错了。
  6. p89.一.2和3 第 2 题答案最后一句位置错了，应该放到第 3 题答案最后。
  7. p89.一.8 答案最后一句解释错误，本题没有哪里出现过 1024。
  8. p91.二.2 这道题里面的 cache/主存系统效率的定义之前从来没看到过，求平均访问时间和求效率两问的顺序不对。
  9. p93 袁教材中描述电容的数据有效保存期上限为 64ms，与王道中的描述的 2ms 相差 32 倍，我们并不能直接把这个 2ms 当成真理，而且我也不认为会直接考这个值，p98.二.2 这道题就很没有意思。谁知道这两个数据的出处在哪里？
  10. p121 倒数第四行，“其他各行计数器均加 1” 应为 “其他小于命中行的各行计数器均加 1”，否则标志位可能溢出。
  11. p137.一.9 第3个说法应该是错的，页面大是缺页代价高而降低操作速度，页面调度频率是不如页面小的情况的。
  12. p143.二.4 解答是错误的，应该由 LRU 替换位标记替换顺序，而不是移动数据。
  13. p153.二.3.3) 的解答中缺少向上取整的符号，R 应该为整数。
  14. p180.一.12 存储器容量（主存容量） -> 主存地址位数-> 程序计数器位数，答案是这个逻辑。但是实际上是程序计数器的位数决定了可寻址空间，因果关系不对。
  15. p187.一.7与 p180.一.11 重复。
  16. p190 最下面 5.3.2 数据通路的基本结构那一块基本上是乱的，应该是想分成内部总线和专用数据通道两类，但是硬生生写了三段。
  17. p208 的注意内容和 p213.一.16 结合起来看很有意思，为什么只有取址、中断要优化成单独的微程序，而间址不需要。
  18. p217.一.18 解答最后一个逗号应该为句号，前面的解释和第四个说法正确没有关系。
  19. p266.二 这道大题假设了每个磁道的容量相等，即内层位密度大于外层位密度，但是实际上现代硬盘并不满足这个假设。也可以假设位密度不变，因此该题目不严谨。
  20. p271.一.7 这道题是选说法不可能的是，而不是说法错误的。
  21. p278 DMA 控制器组成第2各个主要功能中间缺少主语，接管总线控制权的是 DMA 而不是 CPU。
  22. p291.一.40 这里数据应该是送到了 DMA 控制器的数据缓冲寄存器中，而不是 DMA 控制器的 DMAC 中。

更苛刻的要求

  1. 许多选择题中有两个括号，有的只需一个选项，有的需要选两个选项。都比较变扭。p3.一.3、p161.一.24等。
  2. 数据的表示与运算部分逗号和小数点混用。
  3. 部分题目在知识点之前，不合理。p16.二.1，p125.二.6，p139.二.2，p181.一.22
  4. 浮点数的表示范围联考大纲已删除，但是还有大量的习题保留。
  5. 关于 p127 页用蒋本珊教材中的方式，是否有更好的处理方案，而不是故意让同学们做错。
  6. 有的题放错章节了。p138.一.12 应该在上一节。
