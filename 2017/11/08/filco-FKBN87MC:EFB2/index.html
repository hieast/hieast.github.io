Filco 圣手二代87键 Mac 按键适配指南（全是私货）

发表于 2017-11-08 | 分类于 硬件外设 |

前言

2016 款的 MBPR 15 选用了蝶式键盘，键程很短。刚用的时候感觉很硬，敲了一年感觉变软了很多，再用下去可能手感就不能接受了。正好买的转接头上还有多余的 USB 接口，所以打算买一个机械键盘改善工作条件。

非蓝牙、87键以内的小键盘，挑来挑去，我就在 大 F 的圣手二代 和 hhkb pro 2 之间纠结了好几天，最后买了 FKBN87MC/EFB2 黑色青轴入坑。

笔记本型号： 2016款 15寸 MBPR 带 touchbar

操作系统：10.12.6

Mac 适配操作

由于不同的键盘有不同的键位设计，特别是大部分键盘的默认键位都是为 windows 系统进行的适配，直接用在 MBPR 上面会有很多不习惯的地方。有两个方法可以改善。

Keyboard

系统自带的 Keyboard 设置可以配置主要的键盘使用偏好。

个人比较重视的几个：

  1. 按键重复速度设为最快，重复前延迟设为次最短。原因是经常用方向键移动光标，速度不能慢，但是如果重复前延迟时间设置得太短，打字的时候就很容易出现错误的按键重复。
  2. 快捷键，个人习惯 Control + Space 唤醒 Spotlight，Command + Space 切换输入法。
  3. 输入法只保留搜狗拼音（支持微软双拼），最多再加一个美式英语键盘，避免切换输入法的时候选项过多，需要思考。

这里新加了一块 USB 键盘，因此会用到以下功能：

System Preferences -> Keyboard -> Modifier Keys… -> Select Keyboard (choose USB Keyboard) -> switch Option Key and Command Key

由于 macOS 中的 Option 与 Win 10 中的 Alt 对应，Command 与 Win 对应，但是 Win 10 的 Alt 键比 Win 键更靠近 Space 而 macOS 中的 Command 键比 Option 键更接近 Space。因此本人把 macOS 中外接的 USB 键盘对应的 Option 键和 Command 键对调，再把键盘上对应键位的键帽对调即可。

如此键盘基本可以满足 macOS 的使用习惯。但是假如需要更加灵活的配置，更加高级的组合键，那么就需要借助 Karabiner 帮助了。亲测使用 Karabiner 的时候在 Keyboard 中对调的按键会失效，因此在 Karabiner 中需要再次设置。

Karabiner

首先是下载软件，首页 就有很多版本可以下载，不同的软件版本对应不同的系统版本，千万不要下错了。我下载的是 Karabiner-Elements-11.1.0，目前最新的版本，支持10.12 以及更新的 10.13。

软件功能强大，不再赘言，简单说一下我自己的配置。

  1. 将 Option 键和 Command 键对调，这需要四条简单配置。
  2. 将应用 application 键（也有人叫目录键）映射到 fn 键，功能键平时就保持 F1~F12，这样需要时可以组合出全功能键。
  3. 开启右边 command 和 hjkl 组合方向键，将 capslock 换成 右边的 command 键。这样可以在右手不离开主键盘区的情况下控制光标的移动了。缺点是没有 Capslock 键，不过这个键我本来也没用过，中英文切换一直用 shift 足矣。
  4. 右上角的九个按键暂时没用上，有待开发。
